# Darul Sakina

Community & faith organization **web portal** (events, volunteers, donations).

Not e-commerce. Not Rangoons. Not Bijli.

## Run

```bash
git clone https://github.com/badrpk/darulsakina.git
cd darulsakina
npm start
# http://127.0.0.1:8765
```
